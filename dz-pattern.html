Help

Getting Started

Tap anywhere on the map to set a landing target. The app fetches winds aloft for that location and draws a suggested landing pattern. Tap again to move the target.

Final Heading

The heading bar at the bottom controls the final approach direction. By default it points into the surface wind. Drag the slider to override. The yellow pyramid marks the into-wind heading — tap it to snap back.

Pattern

Drifting: base leg is steered 90° to final and wind carries the canopy to the turn point. Traditional: base leg crabs into wind to hold a true 90° ground track. Toggle L/R hand in the heading bar. Pattern altitudes are set in Settings.

Canopy Performance

Enter glide ratio, horizontal speed, and vertical speed — any two calculates the third. Use still-air, no-wind values. Vertical speed equals sink rate (FlySight reports this directly).

Safety Regions (Canopy)

Three amber shaded rings around the pattern entry point show where a canopy pilot must be at each altitude to reach the entry point. They shift upwind to account for drift. Region Buffer shrinks them by a percentage for a safety margin.

Safety Regions (Opening & Exit)

The yellow ring shows where a pilot must open to reach pattern entry under canopy. The blue dashed ring shows where a pilot must exit the aircraft to open within the opening region. Freefall drift is calculated using actual winds aloft at each altitude band.

Jump Run

A suggested jump run is drawn through the exit region center, into the wind at exit altitude. The blue dashed line shows the track. The heading slider in Settings overrides the auto heading — the teal pyramid marks the into-wind position.

Offset: perpendicular distance from the landing spot to the jump run. Shown in muted color when auto-calculated. Enter a value to override (positive = right of flight path).

🟢 Green light: where the jump run enters the exit circle — distance prior to or past the projected landing spot reference. 🔴 Red light: where the jump run exits the exit circle.

Exit separation: suggested seconds between groups based on horizontal separation distance and jump run ground speed.

Winds Aloft

Winds are fetched from Open-Meteo using GFS+HRRR (US) or GFS (international). Teal rows are real model data points. Grey rows are interpolated. The surface wind comes from the HRRR 10m wind at the current hour. Data updates hourly for US locations.

Layers

Use the 👁 button to toggle individual map layers — jump run, exit region, canopy regions, pattern labels, and map legend — independently.

Drop Zone Search

Tap 🔍 to search for a drop zone by name. Results come from the USPA dropzone database. Selecting a DZ sets the map view and landing target to that location.
